# 🎮 Tic Tac Toe

A modern and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. Play against a friend or challenge an AI with multiple difficulty levels. The game includes persistent score tracking, theme switching, sound effects, and smooth animations.

---

## ✨ Features

- 🎯 Single Player (AI) Mode
- 👥 Multiplayer Mode
- 🤖 Three AI Difficulty Levels
  - Easy
  - Medium
  - Hard (Minimax Algorithm)
- 🌙 Dark & ☀️ Light Theme Toggle
- 🔊 Sound Effects
- 🏆 Persistent Scoreboard (Local Storage)
- 🔄 Restart Game
- ♻️ Reset Scores
- 🎉 Winning Cell Animation
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in your browser.

No installation or additional dependencies are required.

---

## 🎮 How to Play

- Choose **Multiplayer** or **Single Player** mode.
- If playing against AI, select the desired difficulty.
- Players take turns placing **X** and **O**.
- The first player to align three symbols horizontally, vertically, or diagonally wins.
- Scores are automatically saved in your browser.
